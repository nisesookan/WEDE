Training Programs

Introduction to Coding

- Content: Basics of programming using languages like Python or JavaScript; fundamental concepts such as variables, loops, and functions.

- *Duration:* 8 weeks (2 hours per week)

- *Enrollment Requirements:* No prior coding experience required; basic computer skills needed.

Web Development Fundamentals

- Content:HTML, CSS, and JavaScript basics; creating and styling web pages; introduction to responsive design.

- Duration: 10 weeks (3 hours per week)

- Enrollment Requirements:Familiarity with basic computer operations; completion of Introduction to Coding is recommended but not required.

Graphic Design and Multimedia

- Content: Introduction to design principles; using tools like Adobe Photoshop and Illustrator; creating graphics, logos, and digital media.

- Duration: 12 weeks (2 hours per week)

- Enrollment Requirements:No prior experience necessary; basic understanding of computer graphics recommended.

Data Analysis and Visualization

- Content:Basics of data analysis using Excel or Google Sheets; introduction to data visualization tools;

creating charts and graphs.

- Duration: 8 weeks (3 hours per week)

- Enrollment Requirements:Basic computer skills; familiarity with spreadsheets is helpful but not essential.

Cybersecurity Basics

- Content: Understanding of cybersecurity principles; basic practices for protecting data and networks; introduction to common threats and security measures.

- Duration:6 weeks (2 hours per week)

- Enrollment Requirements: No prior cybersecurity knowledge required; basic understanding of computers and internet safety is useful.

Advanced Coding and Software Development

- Content: Intermediate to advanced programming concepts; software development methodologies; project-based learning with real-world applications.

- Duration: 14 weeks (4 hours per week)

- Enrollment Requirements: Completion of Introduction to Coding and Web Development Fundamentals, or equivalent experience.

Enrollment Requirements

- General Requirements:Basic computer literacy, including the ability to use a keyboard and mouse, and familiarity with using web browsers and email.

- Specific Requirements: Some advanced courses may require completion of prerequisite courses or equivalent experience.

Enrollment Process

- Application:Complete an online application form available on the YETT website.

- Assessment: For some courses, a brief assessment or interview may be conducted to determine the appropriate level.

- Fee: Most programs are offered for free, but some may require a nominal fee for materials. Scholarships or financial aid options may be available.
